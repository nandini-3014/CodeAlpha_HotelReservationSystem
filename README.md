# Student Grade Tracker

## Description

A Java application to manage student grades and generate reports.

## Features

* Add student names and marks
* Calculate average marks
* Find highest marks
* Find lowest marks
* Assign grades (A, B, C, D, F)
* Generate student report

## Technologies Used

* Java
* OOP (Object-Oriented Programming)
* ArrayList
* Scanner Class

## How to Run

Compile:

javac StudentGradeTracker.java

Run:

java StudentGradeTracker
