# Hotel Reservation System

## Description
A Java-based Hotel Reservation System that allows users to view available rooms, book rooms, cancel bookings, and manage room reservations through a simple console interface.

## Features
- View available rooms
- Book a room
- Cancel a booking
- Display room status
- Menu-driven interface
- Object-Oriented Programming (OOP) implementation

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- Scanner Class

## How to Run

### Compile

```bash
javac HotelReservationSystem.java
